# Neurolution

Mini réseau de neurones qui reconnaît des chiffres (type MNIST).

## Étape actuelle
Setup terminé — environnement Python + PyTorch CUDA opérationnel.

## Prérequis
- Windows 11, GPU NVIDIA RTX 4050 (6 Go VRAM)
- Python 3.14, PyTorch avec CUDA 12.x

## Lancer la vérification GPU
```powershell
.\venv\Scripts\Activate.ps1
python check_gpu.py
```
